This repository contains a WhatsApp chatbot built with FastAPI, integrated with Twilio for WhatsApp messaging and OpenAI GPT-3.5 for intelligent conversational responses.

Features:

Receive WhatsApp messages via a webhook endpoint.

Generate AI responses with OpenAI's GPT-3.5 model.

Send responses back to users through Twilio WhatsApp API.

Easily deployable on Railway.app or any Python-compatible server.

Includes endpoints to test the chatbot without WhatsApp.

This project is ideal for developers looking to build AI-powered conversational bots with minimal setup, using Python and modern APIs.
